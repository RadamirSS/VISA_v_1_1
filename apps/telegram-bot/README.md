# Telegram Bot

Фундамент отдельного Telegram-бота для заявок на поиск записи в визовый центр или консульство.

## Что умеет текущий фундамент

- онбординг и согласие на обработку данных
- регистрация пользователя
- создание заявки на запись
- выбор страны, города подачи и окна поиска
- расчет стоимости по конфигу
- промокоды
- абстракции оплаты и booking provider
- admin workflow для менеджера

## Что пока не делает

- не выполняет реальную запись через внешний API
- не гарантирует наличие слота
- не содержит реальных платежных ключей
- не хранит сканы документов в Telegram

## Локальный запуск

```bash
cd apps/telegram-bot
python -m venv .venv
source .venv/bin/activate
pip install -e .
cp .env.example .env
python -m bot.main
```
