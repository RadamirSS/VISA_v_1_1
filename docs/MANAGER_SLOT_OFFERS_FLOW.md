# Manager Slot Offers Flow

## Как работает процесс

1. Клиент отправляет визовый кейс менеджеру.
2. Менеджер вручную проверяет внешние источники записи вне системы.
3. Менеджер вставляет варианты дат в админ-flow Telegram-бота.
4. Бот валидирует даты и время, сохраняет offer и options.
5. Клиент получает варианты дат в Telegram и видит их в Mini App.
6. Клиент выбирает один вариант.
7. Менеджер получает уведомление о выборе клиента.
8. Менеджер подтверждает финальную запись.

## Ограничения

- в этом PR нет прямой booking API integration
- выбранная клиентом дата еще не означает финальную запись
- финальный статус появляется только после manager confirmation

## Безопасность

В Telegram уведомлениях о слотах допускаются только:

- case id
- telegram username / id
- country
- city
- provider
- date/time

Нельзя отправлять:

- паспортные данные
- место рождения
- адрес
- полную анкету
- документные данные
- банковские данные
