# Privacy And Data Notes

## Website

- собирает только минимальные лид-данные
- не собирает паспортные номера
- не собирает файлы и банковские данные

## Telegram bot

- не собирает сканы документов в чате
- паспортные поля отключены по умолчанию через `ENABLE_SENSITIVE_FIELDS=false`
- если чувствительные поля будут включены, production-версия должна использовать шифрование и безопасное хранение

## Общие принципы

- не коммитить реальные токены и ключи
- не логировать чувствительные значения
- не обещать гарантированную запись, слот или одобрение визы
