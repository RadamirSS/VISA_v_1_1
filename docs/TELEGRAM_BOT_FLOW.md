# Telegram Bot Flow

Бот является отдельным deployable-приложением и главным интерфейсом для workflow по заявкам на поиск записи.

## Основные этапы

1. `/start` и онбординг
2. согласие на обработку данных
3. регистрация пользователя
4. выбор страны
5. выбор города подачи и визового центра
6. выбор цели поездки
7. выбор окна поиска записи
8. выбор количества заявителей
9. ввод данных заявителей
10. промокод
11. расчет цены
12. оплата или подтверждение офлайн-оплаты
13. создание заявки менеджеру

## Admin flow

- просмотр новых заявок
- создание промокодов
- подтверждение оплаты наличными
- ручное изменение статусов

## Ограничения

- mock payment provider
- mock booking provider
- без реальной записи через внешние системы
